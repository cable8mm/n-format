## v1.2.0 - 2020-07-01

We add three methods.

decimal, percent and rawPercent:

```php
<?php

echo NFormat::decimal(12346);

// output
// 12,346

echo NFormat::percent(12346);

// output
// 1,234,600%

echo NFormat::rawPercent(12346);

// output
// 12,346%
```
